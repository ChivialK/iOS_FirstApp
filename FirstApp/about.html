Note: Help

  • Create
  • Edit
  • Delete

Create:
You can create new notes by selecting the + button in the upper right hand side of the main list of notes. This will create a new blank note for editing.

Edit:
You can edit notes by selecting a note from the list on the main list page. This will bring your existing note up in the editor. You can edit the note and save your changes.

Delete:
You can delete notes by selecting the edit button in the upper left corner on the main list page. This will enable the standard list editing function to allow you to delete notes.
